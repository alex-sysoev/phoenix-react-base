    create(conn, arg2)

    Removes user from database

    index(conn, params)

    Returns all users from database

    show(conn, map)

    Return user from database. Could be used to return current user profile

    update(conn, map)

    Updates user using update changeset that does not validate presence of password but if presents hashes it
